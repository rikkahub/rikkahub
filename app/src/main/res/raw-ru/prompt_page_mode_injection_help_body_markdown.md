# Что такое инъекция режима

Инъекция режима — это ручной блок промпта.

Она **не** активируется по ключевым словам, как запись в lorebook, и **не** является полной структурой пресета ST.

Используйте её, когда вам нужен переиспользуемый режим, например:
- режим обучения
- режим перевода
- режим проверки кода
- режим улучшения текста

## Быстрый старт

1. Создайте одну инъекцию.
2. Дайте ей имя, описывающее режим.
3. Выберите позицию рядом с системным промптом.
4. Поместите саму инструкцию в содержимое.
5. Используйте приоритет, чтобы управлять порядком, если несколько инъекций могут быть активны одновременно.

## Пример

```text
Name:
Объясняй просто

Position:
После системного промпта

Priority:
100

Content:
Объясняй понятия простым языком, избегай жаргона и используй один короткий пример, если это уместно.
```

## Как понимать позицию

- **Перед системным промптом**: самое сильное влияние на контекст, обычно для поведения высокого уровня.
- **После системного промпта**: более безопасный вариант по умолчанию для большинства дополнительных инструкций.

> Начните с **После системного промпта**, если только вы не уверены, что вам нужно более сильное переопределение.
