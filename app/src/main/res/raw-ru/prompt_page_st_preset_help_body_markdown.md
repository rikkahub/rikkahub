# Что управляется на этой странице

Это общая **библиотека пресетов SillyTavern** для приложения.

Здесь следует:
- настраивать порядок промптов ST и определения промптов
- управлять runtime-промптами, такими как поведение нового чата и продолжения
- задавать значения сэмплирования
- настраивать regex-скрипты ST

Карточки персонажей для конкретных ассистентов по-прежнему находятся в разделе каждого ассистента.

## Быстрый старт

1. Импортируйте JSON-пресет из SillyTavern или создайте пресет по умолчанию.
2. Откройте библиотеку пресетов, если у вас их много.
3. Выберите один пресет как активный общий пресет.
4. Отредактируйте порядок промптов, runtime-промпты, сэмплирование и regex-правила.
5. Снова экспортируйте, если хотите отправить его обратно в ST.

## Демонстрация: управление большой библиотекой

```text
Цель:
Сделать 20 импортированных пресетов удобными для управления

Шаги:
1. Нажмите «Управление библиотекой»
2. Найдите "claude" или "rp"
3. Выберите нужный пресет
4. Вернитесь на главную страницу и редактируйте только активный пресет
```

## Что означает каждая область

- **Библиотека**: переключение, поиск, экспорт и удаление пресетов.
- **Редактор пресета**: порядок промптов, определения промптов, runtime-тексты, поведение имён.
- **Редактор сэмплирования**: temperature, top-p, штрафы и связанные настройки генерации.
- **Редактор regex**: regex-скрипты в стиле ST, общие для активного пресета.

## Примечание о поведении имён

Здесь приложение сохраняет совместимость с ST. Режим **Content** — это режим, который полностью отражён в текущем runtime приложения.

> Если вы ранее импортировали много пресетов, сначала используйте панель библиотеки. Не пытайтесь управлять всем с главной страницы.
