# Системы промптов в этом приложении

Эта страница разделяет функции промптов, чтобы ими было проще управлять.

## Когда что использовать

- **SillyTavern Preset**: общий порядок ST-промптов, runtime-промпты, sampling и regex-скрипты ST.
- **Mode Injection**: ручной блок промпта, который вы включаете, когда вам нужен режим, например режим перевода или режим кодинга.
- **Lorebook**: информация о мире, вызываемая ключевыми словами и появляющаяся только при обнаружении подходящего контекста.

## Краткое руководство по выбору

```text
Я хочу, чтобы всё приложение использовало одну библиотеку пресетов ST
-> SillyTavern Preset

Я хочу повторно использовать ручной режим, например "Объясняй всё просто"
-> Mode Injection

Я хочу, чтобы факты или лор появлялись только при появлении определённых слов
-> Lorebook
```

## Демо: три похожих запроса, три разных инструмента

### Случай A
"Всегда используй мой импортированный порядок ST-промптов и поведение продолжения"

Используйте **SillyTavern Preset**.

### Случай B
"Когда я переключаюсь в режим обучения, добавляй в начало инструкцию наставника"

Используйте **Mode Injection**.

### Случай C
"Когда в чате упоминается королевская семья, добавляй настройки королевства"

Используйте **Lorebook**.

> Если правило должно срабатывать автоматически от контекста, вероятно, это запись lorebook. Если вы хотите выбирать его вручную, вероятно, это mode injection.
