# 什麼是模式注入

模式注入是一個手動提示區塊。

它**不是**像 lorebook 項目那樣由關鍵字觸發，也**不是**完整的 ST 預設結構。

當你想要可重複使用的模式時，可使用它，例如：
- study mode
- translation mode
- code review mode
- writing polish mode

## 快速開始

1. 建立一個注入。
2. 為它取一個能描述該模式的名稱。
3. 選擇一個靠近系統提示的位置。
4. 將實際指令放入內容中。
5. 如果多個注入可能同時啟用，使用優先順序來控制順序。

## 範例

```text
名稱：
Explain Simply

位置：
系統提示後

優先順序：
100

內容：
用淺白的語言解釋概念，避免術語，並在有幫助時使用一個簡短範例。
```

## 如何理解位置

- **系統提示前**：框架效果最強，通常用於高層級行為。
- **系統提示後**：對大多數額外指令來說，是較安全的預設選擇。

> 除非你知道自己需要更強的覆蓋效果，否則請從 **系統提示後** 開始。
