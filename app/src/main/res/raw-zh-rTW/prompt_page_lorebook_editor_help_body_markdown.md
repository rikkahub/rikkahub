# 編輯 lorebook

此頁面用於管理單一本書及其條目清單。

## 書籍層級欄位

- **名稱**：用於整理。
- **描述**：提供給人類的選填備註。
- **啟用**：開啟或關閉整本書。
- **遞迴掃描**：讓已觸發的條目在後續掃描中啟用更多條目。
- **Token 預算**：限制可從這本書插入的文字量。

## 建議工作流程

1. 建立書籍。
2. 設定是否應使用遞迴。
3. 決定是否需要 Token 預算。
4. 逐一新增條目。
5. 以真實聊天範例測試。

## 示範流程

```text
Book:
School Life

Use case:
僅在出現課堂、社團、宿舍或考試主題時插入校園細節。

Suggested setup:
- Recursive scanning: 先關閉
- Token budget: 先留空
- 為宿舍、學生會、考試規則與社團文化分別新增條目
```

## 何時調整遞迴掃描

只有在某個已觸發的條目應能於後續掃描中喚醒相關條目時，才開啟它。

> 大多數書籍一開始都應保持簡單：關閉遞迴掃描、不設 Token 預算，之後若有需要再收緊設定。
