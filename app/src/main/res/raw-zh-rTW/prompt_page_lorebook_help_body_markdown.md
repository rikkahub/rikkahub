# Lorebook 的用途

Lorebook 是由關鍵字觸發的世界資訊。

每個 Lorebook 都包含許多條目。每個條目都會監看最近的上下文，並在符合條件時，將其內容注入到提示中。

## 適合的使用情境

- 設定事實
- 關係備註
- 派系或地點摘要
- 只應在相關時出現的重複規則

## 快速開始

1. 建立一個 Lorebook。
2. 將它綁定到應使用它的 assistants。
3. 新增含有簡短、明確關鍵字的條目。
4. 撰寫精簡的條目內容。
5. 用真實聊天訊息測試，若觸發過於頻繁，再收緊關鍵字。

## 範例

```text
Lorebook: Kingdom of Vale

條目關鍵字:
vale, royal family, queen elira

條目內容:
Vale 是一個由 Queen Elira 統治的沿海王國。王室重視以外交為先，只有在不得已時才會選擇公開戰爭。
```

## 書籍層級設定

- **遞迴掃描**：已觸發的條目可能會在後續輪次中幫助觸發更多條目。
- **Token 預算**：限制此 Lorebook 最多可提供多少內容。
- **Assistant 綁定**：讓同一個 Lorebook 只在適合的地方共用。

## 實用建議

- 與其用一個龐大的條目，不如使用多個小條目。
- 如果某個關鍵字過於籠統，請加入次要關鍵字或選擇性邏輯。
- 請謹慎使用 **Constant Active**。

> 如果某些內容只應在上下文提到時才出現，Lorebook 通常就是正確的工具。
