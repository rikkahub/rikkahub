# 此頁面管理的內容

這是此 app 共用的 **SillyTavern 預設集資料庫**。

適合放在這裡的有：
- ST 提示詞順序與提示詞定義
- 執行階段提示詞，例如新聊天與繼續行為
- 取樣數值
- ST regex 腳本

助理專用的角色卡仍會放在各助理底下。

## 快速開始

1. 從 SillyTavern 匯入預設集 JSON，或建立預設預設集。
2. 如果你有很多預設集，請開啟預設集資料庫。
3. 選擇一個預設集作為啟用中的共用預設集。
4. 編輯提示詞順序、執行階段提示詞、取樣與 regex 規則。
5. 如果你想把它傳回 ST，請再次匯出。

## 示範：管理擁擠的資料庫

```text
目標：
讓 20 個已匯入的預設集保持好管理

步驟：
1. 點按管理資料庫
2. 搜尋 "claude" 或 "rp"
3. 選擇你要的預設集
4. 回到主頁面，只編輯目前啟用的預設集
```

## 各區域的意思

- **資料庫**：切換、搜尋、匯出與刪除預設集。
- **預設集編輯器**：提示詞順序、提示詞定義、執行階段文字、名稱行為。
- **取樣編輯器**：temperature、top-p、penalties，以及相關的生成設定。
- **Regex 編輯器**：與目前啟用預設集共用的 ST 風格 regex 腳本。

## 名稱行為說明

此處的設計會保留與 ST 的相容性。**Content** 模式是目前 app 內執行階段完整對應的模式。

> 如果你之前匯入了很多預設集，請先使用資料庫抽屜。不要試著從主頁面管理所有內容。
