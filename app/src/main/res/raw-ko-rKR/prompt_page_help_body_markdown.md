# 이 앱의 프롬프트 시스템

이 페이지는 프롬프트 기능을 관리하기 쉽도록 분리해 둡니다.

## 각각 언제 사용하나요

- **SillyTavern Preset**: 공유된 ST 프롬프트 순서, 런타임 프롬프트, 샘플링, 그리고 ST regex 스크립트.
- **Mode Injection**: 번역 모드나 코딩 모드처럼 원하는 모드를 위해 직접 켜는 수동 프롬프트 블록.
- **Lorebook**: 일치하는 컨텍스트가 발견될 때만 표시되는, 키워드로 트리거되는 세계관 정보.

## 빠른 선택 가이드

```text
앱 전체에서 하나의 ST preset 라이브러리를 사용하고 싶다
-> SillyTavern Preset

"모든 것을 쉽게 설명해 줘" 같은 재사용 가능한 수동 모드를 원한다
-> Mode Injection

특정 단어가 나올 때만 사실이나 설정이 나타나게 하고 싶다
-> Lorebook
```

## 예시: 비슷한 세 요청, 서로 다른 세 도구

### 사례 A
"가져온 ST 프롬프트 순서와 계속 동작을 항상 사용해 줘"

**SillyTavern Preset**을 사용하세요.

### 사례 B
"학습 모드로 전환하면, 튜터링 지시문을 앞에 붙여 줘"

**Mode Injection**을 사용하세요.

### 사례 C
"채팅에서 왕실이 언급되면, 왕국 설정을 주입해 줘"

**Lorebook**을 사용하세요.

> 규칙이 컨텍스트에 따라 자동으로 트리거되어야 한다면, 아마 lorebook 항목일 가능성이 높습니다. 직접 수동으로 선택하고 싶다면, 아마 mode injection일 가능성이 높습니다.
