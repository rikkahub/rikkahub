# Lorebook의 기능

Lorebook은 키워드로 트리거되는 세계관 정보입니다.

각 lorebook에는 여러 개의 항목이 들어 있습니다. 각 항목은 최근 컨텍스트를 감시하다가 일치하면 해당 내용을 프롬프트에 삽입합니다.

## 적절한 사용 사례

- 설정 관련 사실
- 관계 메모
- 세력 또는 장소 요약
- 관련 있을 때만 표시되어야 하는 반복 규칙

## 빠른 시작

1. lorebook을 생성합니다.
2. 사용할 assistant에 바인딩합니다.
3. 짧고 구체적인 키워드로 항목을 추가합니다.
4. 간결한 항목 내용을 작성합니다.
5. 실제 채팅 메시지로 테스트한 뒤, 너무 자주 트리거되면 키워드를 더 구체화합니다.

## 데모

```text
Lorebook: Kingdom of Vale

항목 키워드:
vale, royal family, queen elira

항목 내용:
Vale은 Queen Elira가 통치하는 해안 왕국입니다. 왕실은 외교를 우선시하며, 공개 전쟁은 최후의 수단으로만 여깁니다.
```

## 책 수준 설정

- **재귀 스캔**: 트리거된 항목이 이후 패스에서 더 많은 항목을 트리거하는 데 도움이 될 수 있습니다.
- **토큰 예산**: 이 lorebook이 추가할 수 있는 콘텐츠 양의 상한을 정합니다.
- **Assistant 바인딩**: 하나의 lorebook을 필요한 곳에서만 공유되도록 유지합니다.

## 실용적인 조언

- 큰 항목 하나보다 작은 항목 여러 개를 사용하는 편이 좋습니다.
- 하나의 키워드가 너무 포괄적이라면 보조 키워드나 선택적 로직을 추가하세요.
- **Constant Active**는 꼭 필요할 때만 사용하세요.

> 어떤 내용이 컨텍스트에서 언급될 때만 나타나야 한다면, 보통 lorebook이 적절한 도구입니다.
