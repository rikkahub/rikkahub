# 이 페이지에서 관리하는 것

이곳은 앱의 공유 **SillyTavern 프리셋 라이브러리**입니다.

다음 항목을 관리하기에 적합한 곳입니다:
- ST 프롬프트 순서 및 프롬프트 정의
- 새 채팅 및 이어쓰기 동작과 같은 런타임 프롬프트
- 샘플링 값
- ST regex 스크립트

어시스턴트별 캐릭터 카드는 계속 각 어시스턴트 아래에 있습니다.

## 빠른 시작

1. SillyTavern에서 프리셋 JSON을 가져오거나 기본 프리셋을 만드세요.
2. 프리셋이 많다면 프리셋 라이브러리를 여세요.
3. 프리셋 하나를 활성 공유 프리셋으로 선택하세요.
4. 프롬프트 순서, 런타임 프롬프트, 샘플링, regex 규칙을 편집하세요.
5. ST로 다시 보내고 싶다면 다시 내보내세요.

## 데모: 복잡한 라이브러리 관리하기

```text
목표:
가져온 프리셋 20개를 관리 가능하게 유지하기

단계:
1. 라이브러리 관리 탭
2. "claude" 또는 "rp" 검색
3. 원하는 프리셋 선택
4. 메인 페이지로 돌아가 활성 프리셋만 편집
```

## 각 영역의 의미

- **라이브러리**: 프리셋 전환, 검색, 내보내기, 삭제.
- **프리셋 편집기**: 프롬프트 순서, 프롬프트 정의, 런타임 텍스트, 이름 동작.
- **샘플링 편집기**: temperature, top-p, 페널티 및 관련 생성 설정.
- **Regex 편집기**: 활성 프리셋과 공유되는 ST 스타일 regex 스크립트.

## 이름 동작 참고

앱은 여기서 ST 호환성을 유지합니다. **Content** 모드는 현재 앱 내 런타임에 완전히 반영되는 모드입니다.

> 이전에 프리셋을 많이 가져왔다면 먼저 라이브러리 드로어를 사용하세요. 메인 페이지에서 모든 것을 관리하려고 하지 마세요.
