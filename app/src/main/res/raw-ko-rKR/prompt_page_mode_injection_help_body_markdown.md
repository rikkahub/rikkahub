# 모드 인젝션이란

모드 인젝션은 수동 프롬프트 블록입니다.

이것은 로어북 항목처럼 키워드로 트리거되는 방식이 아니며, 전체 ST 프리셋 구조도 아닙니다.

다음과 같은 재사용 가능한 모드가 필요할 때 사용하세요:
- 학습 모드
- 번역 모드
- 코드 리뷰 모드
- 문장 다듬기 모드

## 빠른 시작

1. 인젝션을 하나 만드세요.
2. 모드를 설명하는 이름을 지정하세요.
3. 시스템 프롬프트 근처의 위치를 선택하세요.
4. content에 실제 지시문을 입력하세요.
5. 여러 인젝션이 함께 활성화될 수 있다면 priority로 순서를 제어하세요.

## 예시

```text
Name:
간단히 설명

Position:
시스템 프롬프트 뒤

Priority:
100

Content:
개념을 쉬운 말로 설명하고, 전문 용어는 피하며, 도움이 될 때는 짧은 예시 하나를 사용하세요.
```

## 위치를 이해하는 방법

- **시스템 프롬프트 전**: 가장 강한 프레이밍으로, 보통 상위 수준 동작에 사용됩니다.
- **시스템 프롬프트 후**: 대부분의 추가 지시문에 더 안전한 기본값입니다.

> 더 강한 override가 필요하다는 확신이 없다면 **시스템 프롬프트 후**부터 시작하세요.
