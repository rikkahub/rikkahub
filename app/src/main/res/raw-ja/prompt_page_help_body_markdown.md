# このアプリのプロンプトシステム

このページでは、プロンプト機能を管理しやすいように分けています。

## それぞれを使うタイミング

- **SillyTavern Preset**: 共有されたSTプロンプト順序、ランタイムプロンプト、サンプリング、ST regexスクリプト。
- **Mode Injection**: 翻訳モードやコーディングモードなど、使いたいモードがあるときに手動で有効にするプロンプトブロック。
- **Lorebook**: 一致するコンテキストが見つかったときにのみ表示される、キーワードトリガーの世界設定情報。

## クイック判断ガイド

```text
アプリ全体で1つのSTプリセットライブラリを使いたい
-> SillyTavern Preset

「すべてを簡単に説明する」のような再利用可能な手動モードが欲しい
-> Mode Injection

特定の言葉が出たときだけ事実や lore を表示したい
-> Lorebook
```

## デモ: 似た3つの要望に対する3つの異なるツール

### ケースA
「インポートしたSTプロンプト順序と継続動作を常に使いたい」

**SillyTavern Preset** を使います。

### ケースB
「学習モードに切り替えたとき、先頭に個別指導の指示を追加したい」

**Mode Injection** を使います。

### ケースC
「チャットで王族に触れたとき、王国の設定を挿入したい」

**Lorebook** を使います。

> そのルールがコンテキストから自動的に発動するべきなら、おそらく lorebook エントリです。手動で選びたいなら、おそらく mode injection です。
