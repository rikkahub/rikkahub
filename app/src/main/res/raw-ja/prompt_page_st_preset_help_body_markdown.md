# このページで管理するもの

これはアプリ用の共有 **SillyTavern preset library** です。

ここで管理するのに適しているもの:
- ST のプロンプト順序とプロンプト定義
- 新規チャットや続行動作などのランタイムプロンプト
- サンプリング値
- ST regex スクリプト

アシスタント固有のキャラクターカードは、引き続き各アシスタント配下にあります。

## クイックスタート

1. SillyTavern から preset JSON をインポートするか、デフォルトのプリセットを作成します。
2. プリセットが多い場合は、preset library を開きます。
3. 1 つのプリセットをアクティブな共有プリセットとして選択します。
4. プロンプト順序、ランタイムプロンプト、サンプリング、regex ルールを編集します。
5. ST に戻して送りたい場合は、再度エクスポートします。

## デモ: 混み合ったライブラリを管理する

```text
目的:
インポートした 20 個のプリセットを管理しやすく保つ

手順:
1. 「Manage library」をタップ
2. 「claude」または「rp」で検索
3. 必要なプリセットを選択
4. メインページに戻り、アクティブなプリセットだけを編集
```

## 各エリアの意味

- **Library**: プリセットの切り替え、検索、エクスポート、削除を行います。
- **Preset editor**: プロンプト順序、プロンプト定義、ランタイムテキスト、名前の動作を設定します。
- **Sampling editor**: temperature、top-p、penalties、および関連する生成設定を調整します。
- **Regex editor**: アクティブなプリセットと共有される ST スタイルの regex スクリプトです。

## 名前の動作について

ここではアプリは ST 互換性を維持します。**Content** モードは、現在のアプリ内ランタイムで完全に反映されるモードです。

> 以前に多くのプリセットをインポートしている場合は、まずライブラリドロワーを使ってください。メインページですべてを管理しようとしないでください。
