# 编辑 lorebook

此页面用于管理单个书籍及其条目列表。

## 书籍级字段

- **名称**：用于组织。
- **描述**：给人类看的可选备注。
- **已启用**：打开或关闭整本书。
- **递归扫描**：允许已触发的条目在后续轮次中激活更多条目。
- **Token budget**：限制可从此书中插入的文本量。

## 推荐工作流程

1. 创建书籍。
2. 设置是否应启用递归。
3. 决定是否需要 Token budget。
4. 逐个添加条目。
5. 使用真实聊天示例进行测试。

## 演示工作流程

```text
书籍：
School Life

使用场景：
仅当出现课堂、社团、宿舍或考试相关话题时注入校园细节。

建议设置：
- 递归扫描：一开始关闭
- Token budget：先留空
- 为宿舍、学生会、考试规则和社团文化分别添加独立条目
```

## 何时更改递归扫描

仅当某个已触发的条目应能在后续轮次中唤醒相关条目时，才将其打开。

> 大多数书籍一开始都应保持简单：关闭递归扫描，不设 Token budget，之后如有需要再收紧。
