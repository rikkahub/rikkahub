# 此应用中的提示系统

此页面将提示功能分开，以便于管理。

## 何时使用各项功能

- **SillyTavern Preset**：共享的 ST 提示顺序、运行时提示、采样，以及 ST regex 脚本。
- **Mode Injection**：你手动开启的提示块，因为你想使用某种模式，例如翻译模式或编码模式。
- **Lorebook**：由关键词触发的世界信息，仅在找到匹配上下文时出现。

## 快速决策指南

```text
我希望整个应用使用同一个 ST 预设库
-> SillyTavern Preset

我想要一个可复用的手动模式，例如“用简单方式解释一切”
-> Mode Injection

我希望某些事实或设定只在出现特定词语时显示
-> Lorebook
```

## 演示：三个相似请求，三种不同工具

### 案例 A
“始终使用我导入的 ST 提示顺序和继续行为”

使用 **SillyTavern Preset**。

### 案例 B
“当我切换到学习模式时，在前面添加一条辅导指令”

使用 **Mode Injection**。

### 案例 C
“当聊天提到王室家族时，注入王国设定”

使用 **Lorebook**。

> 如果某条规则应根据上下文自动触发，那它很可能是 lorebook 条目。如果你希望手动选择它，那它很可能是 mode injection。
