# 此页面管理的内容

这里是应用的共享 **SillyTavern 预设库**。

适合在这里管理：
- ST 提示词顺序和提示词定义
- 运行时提示词，例如新聊天和继续行为
- 采样参数
- ST regex 脚本

特定于助手的角色卡仍然在各助手下单独管理。

## 快速开始

1. 从 SillyTavern 导入预设 JSON，或创建一个默认预设。
2. 如果你有很多预设，请打开预设库。
3. 选择一个预设作为当前活动的共享预设。
4. 编辑提示词顺序、运行时提示词、采样参数和 regex 规则。
5. 如果你想将其发送回 ST，请再次导出。

## 演示：管理拥挤的预设库

```text
目标：
让 20 个已导入的预设保持易于管理

步骤：
1. 点击“管理库”
2. 搜索 "claude" 或 "rp"
3. 选择你想要的预设
4. 返回主页面，只编辑当前活动预设
```

## 各区域的含义

- **库**：切换、搜索、导出和删除预设。
- **预设编辑器**：提示词顺序、提示词定义、运行时文本、名称行为。
- **采样编辑器**：temperature、top-p、penalties 及相关生成设置。
- **Regex 编辑器**：与当前活动预设共享的 ST 风格 regex 脚本。

## 名称行为说明

应用会在这里保持与 ST 的兼容性。**Content** 模式是当前应用内运行时能够完整镜像的模式。

> 如果你之前导入了很多预设，请先使用库抽屉。不要尝试在主页面管理所有内容。
